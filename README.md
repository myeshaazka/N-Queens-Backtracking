# N-Queens-Backtracking
Program ini menyelesaikan masalah N-Queens menggunakan algoritma backtracking.

## Cara Kerja
Program menempatkan ratu satu per satu pada setiap baris.
Jika terjadi konflik, program akan melakukan backtracking untuk mencoba posisi lain.

## Cara Menjalankan
python nqueens.py

## Fitur
- Visualisasi CLI
- Menampilkan proses backtracking
